# ppt-skill

This repo contains the `pptx-bojo` Codex skill for creating and editing `.pptx` decks in a constrained Bojo presentation style.

## Install

Copy the `pptx-bojo/` directory into your local Codex skills directory:

```bash
cp -R pptx-bojo ~/.codex/skills/
```

## Contents

- `pptx-bojo/SKILL.md`: skill entry point
- `pptx-bojo/design-system.md`: visual and layout rules
- `pptx-bojo/editing.md`: template-editing workflow
- `pptx-bojo/pptxgenjs.md`: scratch-build workflow
- `pptx-bojo/scripts/`: helper scripts for unpacking, packing, validation, and rendering
